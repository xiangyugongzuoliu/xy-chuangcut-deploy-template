# AGENTS.md

## Overview

xy-chuangcut-deploy-template is a Zeabur deploy template for ChuangCut, an automated video editing tool powered by Google Gemini AI.

## Build & Test

Deploy via Zeabur using the template configuration.

## Boundaries

- Requires Google Gemini API key
- Designed for Zeabur platform deployment
- Supports breakpoint resume, audio-video sync, and multiple narration styles
